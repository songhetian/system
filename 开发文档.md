# 人力智能排班薪资一体化中台 \- 详细开发文档

## 1\. 项目简介

本系统为企业一体化人力中台，覆盖企业组织人员、权限、智能排班、考勤工时、假期管理、薪资核算、报销、培训、可视化拖拽审批流、消息公告、知识库文件预览、Excel批量导入导出、工资条二级密码安全体系。

系统核心设计原则：**统一UI基座、多开源业务逻辑借鉴、零第三方页面硬集成、前后端类型统一、大数据异步化、敏感数据多层安全校验**。

**开源借鉴规则（核心约束）**

- UI 100% 基于 Arco Design Pro 自研统一，不引入任何第三方开源页面样式

- Cal\.com / Twenty / ERPNext / Kimai / Plane 只借鉴：业务逻辑、算法、数据模型、交互流程、产品体验

- 禁止直接复制外部项目页面代码、样式、组件

## 2\. 整体技术选型（最终定稿）

### 2\.1 前端技术栈

- 核心框架：React 18 \+ TypeScript \+ Vite

- UI 基座：Arco Design Pro（唯一UI标准，全局主题定制）

- 状态管理：Zustand（全局轻量状态）

- 服务端状态：React Query（缓存、轮询、异步任务管理）

- 路由：React Router V6

- 数据校验：Zod（前后端统一校验规则）

- 拖拽审批画布：@xyflow/react

- Excel 处理：xlsx \+ file\-saver \+ jszip

- 工具库：dayjs、ahooks

### 2\.2 后端技术栈

- 服务框架：Fastify（高性能、低阻塞）

- ORM：Prisma（强类型、事务、批量操作）

- 数据库：MySQL 8\.0

- 缓存 \& 队列：Redis \+ BullMQ（异步导入导出、任务进度）

- 文件处理：fastify\-multipart、xlsx、archiver

- 定时任务：fastify\-schedule

- 多线程计算：Piscina（大Excel解析防阻塞）

- 对象存储：MinIO（知识库、附件、导出文件）

- 校验：Zod 统一前后端 Schema

### 2\.3 第三方服务

- kkFileView：文档在线预览（Word/Excel/PDF/PPT/图片），实现系统知识库能力

### 2\.4 技术栈补充说明

本栈已经**完整覆盖人力中台所有场景**，无需再新增技术：

- Zod 统一解决表单、Excel、接口三层校验

- BullMQ 解决上万行 Excel、大数据导出超时问题

- MinIO \+ kkFileView 完整支撑知识库体系

- React Query \+ Zustand 解决全局状态与异步任务管理

## 7\. Excel导入导出详细设计

### 7\.1 核心机制

- 前后端 **同一套Zod Schema** 校验

- 小文件（≤5MB/2000行）：同步处理

- 大文件：分片上传 \+ BullMQ异步队列 \+ 进度轮询

- 薪资模块导入导出强制二级密码校验

### 7\.2 能力覆盖模块

人员、排班、考勤、假期、薪资、报销、培训全部支持：模板下载、导入、导出、错误文件回显。

### 7\.3 流程

1. 前端下载后端标准模板

2. 前端表头预校验 \+ 轻量数据校验

3. 分片/同步上传

4. 后端Zod \+ 业务规则双重校验

5. 事务批量入库，失败回滚

6. 返回成功/失败统计 \+ 错误Excel下载

## 8\. kkFileView 知识库集成方案

### 8\.1 部署方式

独立 Docker 容器部署，与业务系统解耦，高可用、不占用业务服务资源。

### 8\.2 对接流程

1. 用户上传文件 → Fastify 接收 → MinIO 存储

2. 生成临时预览URL存入数据库

3. 前端通过统一预览弹窗 iframe 加载 kkFileView

4. 涉密文件预览前触发二级密码校验

### 8\.3 支持文件

PDF、Word、Excel、PPT、图片、TXT，无需下载直接在线预览。

## 9\. 核心交互流程总览

### 9\.1 员工入职全流程

新增员工 / Excel批量导入 → 完善档案 → 分配岗位部门 → 加入排班池 → 考勤自动纳入统计 → 薪资自动核算
